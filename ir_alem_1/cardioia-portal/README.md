# Portal CardioIA - Interface Web

Bem-vindo ao repositório do **Portal CardioIA**, o módulo front-end construído como parte da atividade de Ir Além 1 da Fase 2.

## Sobre a Aplicação
O Portal CardioIA é uma aplicação responsiva e esteticamente agradável que simula o gerenciamento de uma clínica de cardiologia moderna e automatizada com auxílio de IA. Entre as principais features que desenvolvemos, destacam-se:

- **Autenticação Simulada (AuthContext)**: Simulação de um sistema de login com proteção de rotas ativas. Se o usuário não está logado, ele nunca acessa as informações.
- **Consumo de APIs Mocks**: Fetch inteligente baseado no padrão `JSONPlaceholder`, injetado com modelagem randômica para simular dados médicos reais como Risco e Status da triagem.
- **Gerenciamento de Estado**: Sistema de agendamento validado e unificado em complexidade com a integração simultânea de `useState` e `useReducer`.
- **CSS Modules puro**: Efeito Glassmorphism de ponta com estilização local construída do zero, reforçando a imersão em UI/UX e desprendimento de bibliotecas prontas como o Bootstrap.

## Instruções de Instalação e Execução

Necessário possuir o gerador de pacotes **Node.js** instalado na sua máquina (preferencialmente versão 18+).

1. Clone o repositório no seu computador local.
2. Abra o terminal integrado apontando para dentro desta pasta (`cardioia-portal`) e instale as bibliotecas rodando o clássico:
```bash
npm install
```
3. Suba o servidor padrão de desenvolvimento web utilizando o comando ágil do Vite:
```bash
npm run dev
```
4. O terminal fornecerá o link da sua porta local (geralmente `http://localhost:5173/`). Acesse pelo navegador e interaja livremente com o portal!
**Dica de Login**: Use **admin@cardioia.com** com a senha **123456**.
